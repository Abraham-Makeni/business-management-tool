# Business Management Tool

Offline-first business management platform for small and growing businesses.

## MVP Scope
- User login
- Product registration
- Inventory management
- Manual sales checkout
- Cash and M-Pesa payment recording
- Expense tracking
- Daily sales report
- Low stock alerts
- Local backup/export

## Tech Stack
- Python
- PySide6
- SQLite
- SQLAlchemy
- Alembic

## Setup
```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
pip install -r requirements-dev.txt
```

## Run
```bash
python main.py
```
